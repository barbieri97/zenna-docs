# Sessões recorrentes

Para pacientes com horário fixo, agende a série inteira de uma vez usando a **Recorrência**.

## Opções de recorrência

Ao [agendar uma consulta](/agenda/nova-consulta), escolha no campo **Recorrência**:

| Opção | A sessão se repete... |
|---|---|
| **Sem recorrência** | Não se repete — sessão única |
| **Semanal** | Toda semana, no mesmo dia e horário |
| **Quinzenal** | A cada duas semanas |
| **Mensal** | Uma vez por mês |

## Até quando repetir?

Para séries recorrentes, defina o fim de **uma** destas duas formas:

- **Terminar em** — uma data final; ou
- **Número de sessões** — quantas sessões a série terá.

::: warning Contratos pré-pagos exigem um limite
Em contratos de [Sessões Pré-pagas](/contratos/sessoes-pre-pagas), a série **precisa** ter data final ou número de sessões — porque cada sessão da série consome um crédito do pacote, e o Zenna reserva todos de uma vez.
:::

## Sessões com borda tracejada

As próximas sessões de uma série aparecem no calendário com **borda tracejada**. Elas são geradas automaticamente pela recorrência e ainda não foram "confirmadas" individualmente.

Para mexer em uma sessão tracejada (editar, arrastar para outro horário), abra-a e clique em **Confirmar esta sessão** primeiro. Concluir ou alterar a sessão também a torna definitiva automaticamente.

::: tip Isso não muda nada para o paciente
A borda tracejada é só uma indicação visual para você — a sessão está normalmente agendada e aparece nos lembretes.
:::
