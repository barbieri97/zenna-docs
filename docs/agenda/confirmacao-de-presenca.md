# Confirmação de presença

Quando os [lembretes de sessão](/pacientes/lembretes-de-sessao) estão ativados, o paciente recebe um e-mail na véspera com um botão para **confirmar a presença** — sem precisar de senha ou aplicativo.

## Como funciona para o paciente

1. Na véspera da sessão, o paciente recebe o e-mail de lembrete.
2. Ele clica no botão de confirmação.
3. Abre uma página simples mostrando **"Sessão confirmada!"**, com a data e o horário.

## Como funciona para você

Na sua **Agenda**, a sessão muda do status **Agendada** para **Confirmada**. Assim você sabe, antes do horário, quem confirmou.

## Situações possíveis

| Mensagem que o paciente vê | O que significa |
|---|---|
| **Sessão confirmada!** | Tudo certo — a presença foi registrada |
| **Sessão já confirmada** | Ele já tinha clicado antes; nada muda |
| **Link expirado** | O link vale por **3 dias**; a sessão continua agendada, só não dá mais para confirmar por ele |
| **Sessão indisponível** | A sessão foi cancelada ou remarcada — o paciente é orientado a falar com você |
