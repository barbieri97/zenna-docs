# Créditos extras

Os **créditos** alimentam os recursos inteligentes do Zenna, como o [assistente de IA](/documentos/assistente-ia) do editor de documentos. Cada plano inclui uma cota mensal (o Free, por exemplo, inclui 50), renovada a cada ciclo.

## Ver seu saldo

No menu **Assinatura**, o cartão **Plano atual** mostra os **Créditos restantes** do período.

## Comprar créditos extras

Se os créditos do mês acabarem antes da renovação:

1. Abra **Assinatura**.
2. Na seção **Comprar créditos extras**, escolha um dos pacotes.
3. Clique em **Comprar via PIX**.
4. Pague pelo QR Code ou pelo código copia-e-cola.

Os créditos são adicionados à sua conta **após a confirmação do pagamento**.

::: warning CPF necessário
Assim como nos planos pagos, é preciso ter o **CPF** cadastrado no [perfil](/configuracoes/perfil) para comprar créditos.
:::
