# Enviar um arquivo

Guarde no Zenna documentos que você já tem prontos — encaminhamentos recebidos, exames, contratos assinados, qualquer papelada do paciente.

## Passo a passo

1. Abra **Documentos** no menu (ou a aba **Documentos** do paciente) e clique em **Novo Documento** → **Enviar arquivo**.
2. Na janela **Enviar Documento**, arraste o arquivo para a área indicada ou clique para escolher no computador.
3. Preencha:
   - **Título** (obrigatório)
   - **Descrição**
   - **Paciente** — para vincular o arquivo à ficha de alguém
   - **Tags** — etiquetas para organizar (digite e aperte Enter)
4. Clique em **Enviar**.

## Formatos e limite

São aceitos **PDF, imagens (JPG, PNG, GIF), Word e Excel**, com tamanho máximo de **50 MB** por arquivo.

::: tip Use as tags para achar rápido
Etiquetas como "encaminhamento", "exame" ou "contrato" viram filtros na lista de documentos — um clique e você vê só o que interessa.
:::
