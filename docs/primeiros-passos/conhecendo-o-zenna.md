# Conhecendo o Zenna

Um tour rápido pelo menu lateral — tudo o que você usa no dia a dia está a um clique de distância.

## O menu lateral

| Item | O que você faz lá |
|---|---|
| **Dashboard** | Visão do mês: receitas, despesas, sessões de hoje e cobranças pendentes. [Ver guia](/dashboard/) |
| **Pacientes** | Cadastro e ficha completa de cada paciente. [Ver guias](/pacientes/) |
| **Agenda** | Calendário de consultas — dia, semana, mês e lista. [Ver guias](/agenda/) |
| **Finanças** | Cobranças, pagamentos e fluxo de caixa. [Ver guias](/financas/) |
| **Contratos** | Os acordos de cobrança com seus pacientes. [Ver guias](/contratos/) |
| **Assinatura** | Seu plano no Zenna, créditos e histórico de pagamentos. [Ver guias](/assinatura/) |
| **Configurações** | Perfil, dados bancários e tipos de sessão. [Ver guias](/configuracoes/perfil) |

## Rodapé do menu

No canto inferior do menu você encontra:

- **Seu nome e avatar** — identifica a conta conectada, com seu CRP exibido no topo do menu.
- **Botão de tema** — alterna entre claro e escuro.
- **Sair** — encerra a sessão (ícone de porta).

## No celular

Em telas pequenas, o menu vira um botão no topo (☰ **Abrir menu**). Toque nele para abrir o menu lateral; ele fecha sozinho quando você escolhe uma opção.
