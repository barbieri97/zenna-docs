# Agendar uma consulta

## Passo a passo

1. No menu lateral, clique em **Agenda**.
2. Clique em um horário livre no calendário (a data e a hora já vêm preenchidas) — ou use o botão de nova consulta.
3. Preencha o formulário, nesta ordem:
   - **Paciente** — escolha quem será atendido.
   - **Contrato** — o Zenna carrega os contratos ativos do paciente. É o contrato que define como essa sessão será cobrada.
   - **Data** e **Horário**
   - **Recorrência** — deixe "Sem recorrência" para sessão única, ou veja [Sessões recorrentes](/agenda/recorrencia).
   - **Custo da Sessão (R$)** — opcional; use para registrar gastos seus com o atendimento (materiais, supervisão). Esse valor entra como **despesa** no seu fluxo de caixa quando a sessão é realizada.
4. Salve.

Você também pode agendar direto da ficha do paciente, pelo botão **Nova Sessão**.

::: warning "Nenhum contrato com sessões disponíveis"
Se essa mensagem aparecer ao escolher o paciente, é porque nenhum contrato dele permite agendar agora. Dois motivos comuns:
- Um contrato de [Avaliação Fixa](/contratos/avaliacao-fixa) já atingiu o número de sessões.
- Um contrato de [Sessões Pré-pagas](/contratos/sessoes-pre-pagas) está sem créditos — gere o débito do pacote ou verifique o saldo.
:::
