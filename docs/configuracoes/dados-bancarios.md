# Dados bancários

O cartão **Dados Bancários**, em **Configurações**, guarda as informações de recebimento — principalmente a sua **chave PIX**.

## Campos

- **Titular da conta**
- **Chave PIX** — CPF, e-mail, telefone ou chave aleatória
- **Banco**
- **Tipo de conta** — Conta Corrente ou Conta Poupança
- **Agência** e **Conta**
- **Informações adicionais**

Preencha e clique em **Salvar dados bancários**.

::: tip A chave PIX liga o financeiro inteiro
É a chave cadastrada aqui que gera o **QR Code e o código copia-e-cola** nas suas cobranças. Sem ela, o botão PIX das cobranças fica indisponível. Veja [Receber com PIX](/financas/pix).
:::
