# Criar sua conta

Criar uma conta no Zenna é gratuito e leva menos de um minuto.

## Passo a passo

1. Abra a página de acesso do Zenna.
2. Clique em **Cadastre-se** (abaixo do botão de entrar).
3. Preencha os campos:
   - **Nome** e **Sobrenome**
   - **E-mail** — será o seu login
   - **Senha** e **Confirmar senha**
4. Clique em **Criar conta**.

Pronto! Você entra direto no seu painel.

::: warning Senha com no mínimo 12 caracteres
Por segurança, a senha precisa ter **pelo menos 12 caracteres**. Uma dica: use uma frase fácil de lembrar, como `meu-consultorio-2026`.
:::

## O que você ganha ao criar a conta

Sua conta começa no plano **Free**, que permite até **5 pacientes** e inclui **50 créditos por mês** para o assistente de IA. Quando o consultório crescer, é só [mudar de plano](/assinatura/mudar-de-plano).
